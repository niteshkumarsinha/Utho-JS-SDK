Installation

Install the full SDK or specific service packages via your favorite package manager.

MONOLITHIC INSTALLATION
npm install utho-sdk-js
MODULAR INSTALLATION (RECOMMENDED)
npm install @utho/sdk-core @utho/sdk-objectstorage

Quickstart

Initialize the SDK and start making requests in seconds.

MONOLITHIC USAGE
const Utho = require('utho-sdk-js');

const utho = new Utho('YOUR_API_KEY');

async function main() {
    const servers = await utho.cloudserver.list();
    console.log(servers);
}

main();
MODULAR USAGE (RECOMMENDED)
const Client = require('@utho/sdk-core');
const ObjectStorage = require('@utho/sdk-objectstorage');

const client = new Client('YOUR_API_KEY');
const os = new ObjectStorage(client);

async function main() {
    const buckets = await os.listBuckets('in-mumbai-1');
    console.log(buckets);
}
main();

Authentication

The Utho API uses API Keys for authentication. You can generate one from the Utho console.

Note: Keep your API key secret. Never expose it in client-side code or public repositories.
